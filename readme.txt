== Description ==

Business Roy is a perfect and professional advance feature-rich multipurpose business free WordPress theme. Basically, Business Roy multipurpose theme helps you to create different types of business websites ( consultant, finance, corporate, freelancer, agency, business promotion, electrician, industries, education, fashion, health & medical, wedding, photography, gym, architecture, lawyer & hotel ) and many for other websites with the helps theme default features and Elementor custom addon, theme is fully compatible with the latest page builders plugins ( Elementor, SiteOrigin, Visual Composer ) also by default created 10+ different Elementor addon ( Widget ) which you can easily implement in any page or post to create, edit and update beautifully page design, also theme has some exciting features list like ( section re-order every homepage section, translation ready fully compatible with (WPML & Polylang ) plugins, beautifully 2 different blog layout, well manage post format ( gallery, video, audio, quote, normal image ), site speed optimized,  child theme support, quality code, responsive, cross-browser compatible, SEO friendly and supports WooCommerce and some other external plugins like Elementor, SiteOrigin Page Builder, Jetpack, Polylang, Yoast SEO, Contact Form 7 and many more plugins.


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
